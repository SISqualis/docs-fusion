Como acesso o Fusion?

O sistema Fusion é uma plataforma robusta, projetada para ser implementada diretamente na infraestrutura do cliente, garantindo segurança e controle sobre os dados. Esta seção detalha o procedimento padrão para acessar o sistema e as opções disponíveis na tela de login, incluindo a recuperação de senha.

Procedimento de Login​

Para acessar a plataforma Fusion, siga os passos abaixo:

  • 🖥️ Desktop

Acesso via Computador:

🌐

1. Abra o Navegador

Utilize Chrome, Firefox, Edge ou Safari

🔗

2. Insira o Endereço

fusion.nomedohospital.com.br/login

🔐

3. Autenticação

Insira usuário e senha nos campos apropriados

Dica Importante

Para garantir um acesso estável e seguro, é crucial que suas credenciais estejam sempre atualizadas. Além disso, verifique se a sua conexão com a rede local da instituição está configurada corretamente.

🔧 Configurações Avançadas de Rede

Requisitos de Rede:

  • Conexão estável à rede interna da instituição

Troubleshooting:

  • Verifique se está conectado à rede correta
  • Limpe o cache do navegador se necessário
  • Entre em contato com o TI em caso de problemas persistentes

Recuperação de Senha​

🔑

Esqueceu sua senha? Não se preocupe!

Nosso sistema oferece um processo seguro e rápido para recuperação de credenciais.

Caso você tenha esquecido sua senha, o sistema Fusion oferece um mecanismo para recuperá-la:

  1. Localize o Link: Na tela de login, procure pelo link ou botão destinado à recuperação de senha (geralmente intitulado "Esqueceu sua senha?", "Lembrar senha" ou similar).
  2. Informe seu E-mail: Ao clicar no link, você será direcionado para uma nova tela. Nesta tela, insira o endereço de e-mail associado ao seu cadastro no sistema.
  3. Receba as Instruções: Após submeter seu e-mail, o sistema enviará instruções detalhadas para o endereço fornecido, permitindo que você redefina ou recupere sua senha.

📧

Tempo médio

2-5 minutos

🔒

Segurança

Criptografia AES-256

🕐

Validade

Link válido por 24h

Importante

Mantenha suas credenciais sempre seguras e nunca compartilhe com terceiros. O sistema Fusion registra todos os acessos para auditoria de segurança.

Pronto para começar?

Este guia simplificado visa facilitar seu primeiro contato e o acesso contínuo ao sistema Fusion.
