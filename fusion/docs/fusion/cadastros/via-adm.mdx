# Via Administração

O módulo **Via Administração** permite cadastrar as diferentes formas pelas quais medicamentos ou substâncias são administrados aos pacientes, como via oral, endovenosa, tópica, entre outras. Esse cadastro é essencial para garantir precisão nas prescrições e compatibilidade com sistemas integrados.

---

## Acessando o Cadastro

1. No menu lateral, clique em **Cadastros**.
2. Selecione a opção **Via Administração**.

A tela apresenta uma lista com as vias já cadastradas, contendo:

- **Id**: Identificador único.
- **Nome**: Nome da via de administração (ex: ORAL, EV - ENDOVENOSA, INALATÓRIA).

![Listagem de Vias de Administração](/img/lista_via_adm.png)

---

## Criando ou Editando uma Via

Para **adicionar** uma nova via, clique em **+ Nova Via de Administração**.  
Para **editar**, clique no botão **Editar** ao lado do item desejado.

<Admonition type="info" title="Campos de Cadastro">
- **Nome**: Nome da via de administração utilizada na prescrição.
- **Códigos de De/Para**: Mapeamentos para integração com sistemas externos como MV, Tasy, HMED, entre outros.
</Admonition>

![Edição de Via de Administração](/img/edit_via_adm.png)

---

## Dicas e Cuidados

<Admonition type="tip" title="Boas Práticas">
- Utilize siglas padronizadas e amplamente reconhecidas (ex: EV para Endovenosa, VO para Via Oral).
- Preencha corretamente os códigos de integração conforme exigido pelos sistemas externos.
- Evite duplicidades com variações desnecessárias (ex: "ORAL" e "VO - ORAL").
</Admonition>
