# Anexos

O módulo **Anexos** permite cadastrar os tipos de arquivos aceitos e utilizados no sistema, como PDFs, imagens, entre outros. Esse cadastro é essencial para garantir que os documentos recebidos ou exportados sigam um padrão compatível com o ambiente da instituição e com sistemas integrados.

---

## Acessando o Cadastro

1. No menu lateral, clique em **Cadastros**.
2. Selecione a opção **Anexos**.

A tela exibirá a lista com os tipos de anexo cadastrados, contendo:

- **Id**: Identificador único.
- **Nome**: Nome descritivo do tipo de anexo (ex: PDF).

![Listagem de Anexos](/img/lista_anexos.png)

---

## Criando ou Editando um Tipo de Anexo

Para **adicionar** um novo tipo, clique em **+ Novo Tipo de Anexo**.  
Para **editar**, clique no botão **Editar** ao lado do item desejado.

<Admonition type="info" title="Campos de Cadastro">
- **Nome**: Nome do tipo de anexo.
- **Extensão**: Extensão do arquivo que será aceita (ex: .PDF, .JPG).
- **Códigos de De/Para**: Utilizados para integração com outros sistemas, como MV, Tasy, HMED, etc.
</Admonition>

![Edição de Tipo de Anexo](/img/edit_anexos.png)

---

## Dicas e Cuidados

<Admonition type="tip" title="Boas Práticas">
- Cadastre apenas extensões permitidas e seguras para o ambiente hospitalar.
- Mantenha os nomes claros e descritivos para facilitar a identificação dos tipos.
- Garanta compatibilidade com os sistemas externos por meio dos códigos de integração.
</Admonition>
