# Tipo de Endereço

O módulo **Tipo de Endereço** permite cadastrar e classificar os diferentes tipos de endereços utilizados no sistema, como residencial, principal ou padrão. Essa categorização auxilia na organização de dados de contato e é essencial para integrações com sistemas externos.

---

## Acessando o Cadastro

1. No menu lateral, clique em **Cadastros**.
2. Selecione a opção **Tipo de Endereço**.

A tela exibirá a lista com os tipos já cadastrados, contendo:

- **Id**: Identificador único.
- **Nome**: Nome do tipo de endereço (ex: RESIDENCIAL, Principal, Padrão).

![Listagem de Tipos de Endereço](/img/lista_tipo_endereco.png)

---

## Criando ou Editando um Tipo de Endereço

Para **adicionar** um novo tipo, clique em **+ Novo Tipo de Endereço**.  
Para **editar**, clique no botão **Editar** ao lado do item desejado.

<Admonition type="info" title="Campos de Cadastro">
- **Nome**: Nome do tipo de endereço.
- **Códigos de De/Para**: Campos destinados ao mapeamento com sistemas externos como MV, Tasy, HMED, etc.
</Admonition>

![Edição de Tipo de Endereço](/img/edit_tipo_endereco.png)

---

## Dicas e Cuidados

<Admonition type="tip" title="Boas Práticas">
- Use nomes claros e consistentes para facilitar a identificação do endereço principal do paciente.
- Certifique-se de preencher corretamente os campos de De/Para para garantir a compatibilidade com sistemas integrados.
- Evite criar tipos redundantes com grafias diferentes (ex: "residencial" e "RESIDENCIAL").
</Admonition>
