# Unidades de Medida

O módulo **Unidades de Medida** permite o cadastro das unidades utilizadas para registros clínicos, prescrições e integrações com sistemas externos. Isso inclui medidas de massa, volume, temperatura, frequência, entre outras.

---

## Acessando o Cadastro

1. No menu lateral, clique em **Cadastros**.
2. Selecione a opção **Unidades de Medida**.

A listagem apresenta os registros com os campos:

- **Id**: Identificador único.
- **Sigla**: Abreviação da unidade (ex: kg, cm, mmHg).
- **Nome**: Descrição da unidade (ex: Quilogramas, Centímetros).

![Listagem de Unidades de Medida](/img/lista_medidas.png)

---

## Criando ou Editando uma Unidade

Para **adicionar** uma nova unidade, clique em **+ Nova Unidade de Medida**.  
Para **editar**, clique no botão **Editar** ao lado do item correspondente.

<Admonition type="info" title="Campos de Cadastro">
- **Sigla**: Forma abreviada da unidade (ex: kg, ml, bpm).
- **Nome**: Nome completo da unidade.
- **Códigos de De/Para**: Mapeamentos para sistemas como MV, Tasy, HMED, entre outros.
</Admonition>

![Edição de Unidade de Medida](/img/edit_medidas.png)

---

## Dicas e Cuidados

<Admonition type="tip" title="Boas Práticas">
- Use siglas padronizadas internacionalmente.
- Mantenha a lista limpa, evitando duplicidades ou variações com grafia diferente.
- Certifique-se de preencher os códigos de integração corretamente.
</Admonition>
