# Status de Solicitação de Procedimento

O módulo **Status de Solicitação de Procedimento** permite controlar a situação de uma solicitação médica ou assistencial registrada no sistema, como “Solicitado”, “Agendado”, “Cancelado”, entre outros. Essa funcionalidade garante rastreabilidade e organização dos fluxos de atendimento.

---

## Acessando o Cadastro

1. No menu lateral, clique em **Cadastros**.
2. Selecione a opção **Status de Solicitação de Procedimento**.

A listagem exibirá os status já cadastrados, com os campos:

- **Id**: Identificador único.
- **Nome**: Nome do status (ex: SOLICITADO, CANCELADA, REALIZADA).

![Listagem de Status de Solicitação de Procedimento](/img/lista_stt_sol_proced.png)

---

## Criando ou Editando um Status

Para **adicionar** um novo status, clique em **+ Novo Status de Solicitação de Procedimento**.  
Para **editar**, clique no botão **Editar** ao lado do item desejado.

<Admonition type="info" title="Campos de Cadastro">
- **Nome**: Situação da solicitação de procedimento.
- **Códigos de De/Para**: Usado para integrações com sistemas externos como MV, Tasy, HMED, etc.
</Admonition>

![Edição de Status de Solicitação de Procedimento](/img/edit_stt_sol_proced.png)

---

## Dicas e Cuidados

<Admonition type="tip" title="Boas Práticas">
- Utilize status claros e objetivos (ex: “PENDENTE”, “REALIZADA”, “CANCELADA”).
- Mantenha os códigos de De/Para devidamente preenchidos para integração correta.
- Evite duplicidade de status com pequenas variações de nomenclatura.
</Admonition>
