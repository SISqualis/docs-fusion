# Manual de Configuração do Acesso ao PEP MV via URL

<Admonition type="info" title="Objetivo">
Este documento tem como objetivo padronizar e detalhar o processo de configuração do acesso de outras ferramentas ao Prontuário Eletrônico do Paciente (PEP) MV via chamadas de URL com passagem de parâmetros.

Ele fornece orientações técnicas para integração, define os parâmetros necessários, garante segurança e controle de acesso, além de apresentar exemplos práticos e tratamento de erros. Dessa forma, busca facilitar a comunicação entre sistemas e assegurar um acesso eficiente e padronizado via PEP MV.

</Admonition>

---

## Configuração

### Passo 1 – Criação de Segmento no Perfil do PEP

No perfil do PEP, é necessário criar uma **entrada do tipo Seguimento em Módulo**.

![Criação de Seguimento](/img/pep_segmento.png)

---

### Passo 2 – Cadastro da URL no Segmento

No cadastro do segmento criado, deve-se incluir a **URL com os parâmetros necessários**.

![Cadastro de URL](/img/pep_url.png)

No caminho, informe os seguintes parâmetros:

* `MVPEP_SUBSEGMENTO` = MEVO
* `MVPEP_LINK_URL`

![Cadastro de Parâmetros](/img/pep_param.png)

<Admonition type="tip" title="Exemplo de Caminho">

```plaintext
DS_CAMINHO: http://mevoapp.hpp.br:8080/prescricao/criar/<CDATENDIMENTO>/<CDUSUARIO>/bWV2bzptZXZvQEJRcEhzd0U1eHNHaA==
sn_recarrega_modulo: 'S'
```

</Admonition>

Essas informações garantem a passagem correta dos dados e a execução segura do acesso ao prontuário do paciente através de token via URL integrada ao PEP MV.
